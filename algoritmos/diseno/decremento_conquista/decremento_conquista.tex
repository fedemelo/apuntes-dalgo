\section{Decremento y conquista}

El paradigma de \concept{decremento y conquista} aprovecha la relación entre el problema y una versión más pequeña del problema. La idea es imaginar que la versión más pequeña del problema ya está resuelta y aprovechar eso para terminar de resolver el problema.

La versión más pequeña se obtiene reduciendo el problema a distintas tasas:
\begin{itemize}
    \item \concept{Decremento por una constante}: se reduce el tamaño del problema en una cantidad fija en cada iteración del algoritmo, usualmente en uno.
    \item \concept{Decremento por un factor constante}: se reduce el problema en una proporción fija en cada iteración, usualmente a la mitad.
    \item \concept{Decremento de tamaño variable}: se reduce el tamaño en distinta cantidad en cada iteración.
\end{itemize}

Se presentan tres algoritmos para concretar esta idea, cada uno corresponde a una variante de decrecimiento y conquista.
